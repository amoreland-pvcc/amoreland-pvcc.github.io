Chapter 6: Programming Language

machine language a system that encodes sequences of instructions that are encoded as numerical digits
debugging a tedious task that often leads to errors that must be located and corrected before the job is finished
assembler programs used to convert mnemonic expressions into machine language instructions
assembly language a mnemonic system for representing programs
translator a program that translated programs expressed in high-level primitives into machine-language programs
compiler a translator that often has to compile several machine instructions into short sequences to simulate the activity requested by a single high-level primitive
interpreter similar to translators (an alternative to translators) except that they executed the instructions as they were translated instead of recording the translated version for future use
cross-platform software software that is independent of an operating system’s design as well as the machine’s hardware design and is therefore executable throughout a network
natural language the languages people speak, such as English, Spanish, and French
formal language languages that are designed by people for specific applications
machine independence machines that do not rely on the characteristics of a particular machine
high-level programming language languages that allow locations in main memory to be referenced by descriptive names rather than by numeric addresses
object-oriented programming (OOP) a programming process that uses a software system which is capable of performing the actions that are immmediately related to itself as well as requesting actions of other objects
scripting language a subset of the imperative programming languages
data types:
integer numeric data consisting of whole numbers, probably stored using two’s complement notation
float numeric data that might contain values other than whole numbers, probably stored in floating-point notation
string used for storing text/characters
------------------------------------- -------------------------------------------------------------------------- -------------------------------------------------------------------------- --------------------
primitive data type data types that are included as primitives in a programming language, such as int for integer and char for character
data structure the conceptual shape or arrangement of data; For example, text is normally viewed as a long string of characters, whereas sales records might be envisioned as a rectangular table of numeric values
Python collection data structures:
list an ordered collection of items, and it is one of the essential data structures when using Python to create a project; “ordered collections” means that each item in a list comes with an order that uniquely identifies them
tuple a built-in data structure in Python that is an ordered collection of objects
set an unordered collection with no duplicate elements
dictionary Python’s implementation of a data structure that is more generally known as an associative array. A dictionary consists of a collection of key-value pairs
------------------------------------- ---------------------------------------------------------------------------- ---------------------------------------------------------------------------- ----------------
function a set of instructions for performing a task that can be used as an abstract tool by other program units
local variable a variable declared within a function
global variable variables whose scopes are not restricted to a particular part of a program
source program the program in its original form before being translated
object program the program after it has been translated; the translated version
