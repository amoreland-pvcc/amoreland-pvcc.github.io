Chapter 4 - Networking and the Internet

Network Classifications
LAN a computer network that consists of a collection of computers in a single building or building complex
MAN a network of intermediate size, such as one spanning a local community
WAN a network that links machines over greater distance -- perhaps in neighboring cities or on opposite sides of the world
Network Models
bus network a classification of network based on topology where the machines are all connected to a common communication line; ex. ethernet
star network a classification of network based on topology where one machine serves as a central focal point to which all the others are connected; ex. wireless networks
Protocols
CSMA/CD Ethernet a protocol with the right to transmit messages to all the machines on the bus, where each machine monitors all the messges but it only keeps those addressed to itself
CSMA/CD Wifi a protocol which is compatible with wireless networks, where the policy is to avoid colliding with the transmissions of another machine; a collision avoidance protocol
Combining networks
repeater a device that passes signals back and forth between the two original buses without considering the meaning of the signals
bridge a more complex version of the repeater, which connects two buses but does not necessarily pass all messages across the connection
switch essentially a bridge with mutiple connections, allowing it to connect several buses rather than just two
router devices that handle the connection between networks to form an internet
gateway the "point" at which one network is linked to an internet
Client/server model
P2P: peer-to-peer a way of distributing files so that one peer may receive a file from another and then provide that file to other peers; more broadly, a system by which two processes comminicate over a network
distributed system systems thay consist of software units that execute as processes on different computers; ex. cluster computing in which many independent comptuers work closely together to provide computation or services comparable to a much larger machine
cloud computing a system that allows huge pools of shared computers on the network ban be allocated for use by clients as needed
Internet Addressing
Tier-1 ISP an internet service provider that consists of very high-speed, high-capacity, international WANS; thought of as the backbone of the internet
Tier-2 ISP an internet service provider that tend to be more regional in scope and less potent in their capabilities; along with tier-1, a network of routers that collectively provide the interney's communication infrastructure
Tier-3(access) ISP an independent internet, sometimes called an intranet, oerated by a single authority that is in the business of supplying internet access to indivual homes and businesses
IP address
ICANN a nonprofit corporation established to coordinate the Internet's operation icann.org
DNS a naming database in which internet domain names are located and translated into IP addresses
TLD: Top-level domain the suffix following the period used to relfect the domain's classification; ex. .edu or .org
Country-code TLDs two-letter version of a TLD used for specific countries worldstandards.eu
Internet protocols
FTP (file transfer protocol) a standard protocol for the transfer of files between computer sustems from one client to one server over a TCP/IP network
HTTP an application protocol used for data communication and is the base of data communication on the World Wide Web
SMTP a way that two computers on the network may interact when transmitting an email message from one host to the other
World Wide Web
URL a unique address given to each document on the web that is used to locate and retrieve said document
client-side activities the activity that occurs on the user's computer; also known as front end
server-side activities the activity that occurs on a server and generates dynamic HTML pages that are sent to the client
Internet four-layer software
application layer Layer of the TCP/IP protocol stack that sends and receives data for particular applications such as DNS, HTTP, and SMTP.
transport layer responsible for facilitating host-to-host communication between applications on two different hosts
network layer the internet protocol layer; defines the end points of the packet where the packet starts at one computer and the destination of the packet at the other end
link layer the lowest layer in the internet protocol suite, the networking architecture of the Internet
Internet is...
packet-switching network a network classified by the transfer of small pieces of data across various networks that allow for faster, more efficient data transfer
fault tolerant the ability of a system to continue operating without interruption when one or more of its components fail
redundant the duplication fo critical components or functions of a system with the intention of increasing reliablility of the system; ex. back-up or fail safe
Cybersecurity
malware malicious software that can be used to attack a computer system and its contents
virus a software that infects a computer by inserting itself into programs that already reside in the machine
worm an automous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
Trojan horse a program that enters a comptuer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim
spyware software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
phishing a technique of obtaining information explicitly by simply asking for it
DDoS attack the process of overloading a computer with messages, stands for "Distributed Denial of Service"
Protection
firewall a primary prevention technique used to filter traffic by passing it through a point in the network
spam filter a variation of a firewall, which uses rather sophisticated techniques to distinguish between desirable email and spam
proxy server a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
antivirus software a software which is used to detect and remove a specific type of infection
cryptography the process to secure information and communication techniques derived from mathematical concepts and a set of rule-based calculations called algorithms, to transform messages in ways that are hard to decipher
public key encryption a type of encryption which involves techniques by which encryption systems are designed so that having knowledge about how messages are encrypted does not allow one to decrypt messages
